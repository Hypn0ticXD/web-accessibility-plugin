# web-accessibility-plugin

Šis projekts ir tīmekļa pieejamības paplašinājums, kas izstrādāts, lai uzlabotu vietņu lietojamību lietotājiem ar dažādām vajadzībām, tostarp redzes traucējumiem un lasāmības grūtībām. Paplašinājums dinamiski uzlabo tīmekļa lapu izskatu un struktūru, piedāvājot tādas funkcijas kā fonta izmēra pielāgošana, kontrasta maiņa, pelēktoņu režīms un saišu izcelšana.

Risinājums ir izstrādāts, izmantojot JavaScript, un darbojas, reāllaikā modificējot dokumenta objekta modeli (DOM) un piemērojot CSS stilus. Lietotāji var izmantot vienkāršu saskarni, lai pielāgotu pārlūkošanas pieredzi atbilstoši savām vajadzībām.

Šo paplašinājumu iespējams integrēt tīmekļa projektos vai izmantot kā pārlūkprogrammas paplašinājumu, nodrošinot vieglu veidu, kā uzlabot pieejamību bez nepieciešamības mainīt sākotnējo vietnes kodu.
